# 3bit Emulator

Multi-cycle 3bit program executor, with IF, EX, and WB stages
- [x] end-to-end runnable
- [x] unit tests
- [x] pretty printer
- [x] parse from cli

Run by calling e.g. `sbt "run example.3bit"`

Pretty printer inspired by: https://gist.github.com/Mark1626/1221c1750e38d55fe605225cf877356c
